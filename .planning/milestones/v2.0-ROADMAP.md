# Milestone v2.0: Production Platform

**Status:** ✅ SHIPPED 2026-03-17
**Phases:** 1–4
**Total Plans:** 17

## Overview

Transform the CLI-only sequential orchestration engine (v1.3.0) into a type-safe parallel execution platform with full web interface. The milestone spans 4 phases: Pydantic v2 type safety, asyncio parallel execution with DAG scheduling, FastAPI web dashboard with JWT auth and WebSocket real-time updates, and Experience Store with backward compatibility and CI/CD production hardening.

**Key metric:** 4.65x parallel speedup validated | 467 tests passing | 14,275 LOC Python | ~128 commits over 24 days (2026-02-22 → 2026-03-17)

## Milestone Summary

**Key Decisions:**
- FastAPI over Django REST Framework — lighter, async-native, better WebSocket support
- HTMX/Alpine.js over React — no build step, SSR-friendly, sufficient for dashboard
- StatelessCoordinator (Pure Function Architecture) — eliminates global state, multi-user safe
- JWT + refresh rotation over sessions — stateless, API-friendly, rotation prevents replay attacks
- SQLite WAL over PostgreSQL — sufficient for single-host, no infrastructure dependency
- sentence-transformers (local) over OpenAI embeddings — no API cost, runs in CI

**Issues Resolved:**
- db_path not propagating to routes (dependency_overrides fix in create_app)
- jti collision bug in JWT tokens (same-second refresh causing rotation failure)
- Pyright errors from flow_config annotation, Any params, JSONEncoder signature
- Docker static files not serving (WEB_DIR path resolution in container)

**Issues Deferred:**
- --parallel flag missing in `orchestrate run` CLI command (v2.1)
- trufflehog secret scanner (long build time, manual install needed)
- 3 coordinator tests with timestamp comparison flakiness (non-critical)

**Technical Debt Incurred:**
- YAML export implementation details incomplete (Phase 3 warning, not blocking)
- SECRET_KEY hardcoded (TODO: load from ENV_VAR — marked in code)
- PRP-00-00 Pure Function Architecture tasks 4-10 not completed (optional refactor)
