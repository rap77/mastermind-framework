Create Zustand store for managing scrubber state and snapshot navigation:

1. **ReplayState interface:**
   ```typescript
   interface SnapshotMilestone {
     index: number;
     timestamp: number;
     label: string;
     brainCount: number;
   }

   interface Snapshot {
     timestamp: number;
     snapshot: Map<string, BrainState>; // brain_id → BrainState
   }

   interface ReplayState {
     snapshots: Snapshot[];
     milestones: SnapshotMilestone[];
     currentSnapshotIndex: number;
     taskId: string | null;

     // Actions
     setSnapshots: (snapshots: Snapshot[]) => void;
     computeMilestones: () => void;
     jumpToMilestone: (index: number) => void;
     getCurrentSnapshot: () => Map<string, BrainState> | null;
     getScrubberPercentage: () => number;
     reset: () => void;
   }
   ```

2. **Implementation:**
   - Use `immer` middleware from Zustand (already in Phase 05 setup)
   - enable MapSet plugin (required for Map iteration in Immer)
   - Milestones computed from snapshots using "max 7 milestones" rule (Miller's Law chunking)
   - Milestone interval = Math.ceil(snapshots.length / 7)

3. **Store logic:**
   ```typescript
   const computeMilestones = (snapshots: Snapshot[]) => {
     if (!snapshots.length) return [];
     const interval = Math.ceil(snapshots.length / 7);
     const milestones: SnapshotMilestone[] = [];

     for (let i = 0; i < snapshots.length; i += interval) {
       const snapshot = snapshots[i];
       const activeBrains = Array.from(snapshot.snapshot.values())
         .filter(b => b.status !== 'idle').length;

       milestones.push({
         index: i,
         timestamp: snapshot.timestamp,
         label: `${Math.round((i / snapshots.length) * 100)}% (${activeBrains} active)`,
         brainCount: activeBrains,
       });
     }
     return milestones;
   };
   ```

4. **Performance:**
   - snapshots array stored as-is (JSONB from backend, parsed on frontend)
   - currentSnapshotIndex is number (O(1) lookup)
   - No re-computation on every update (computeMilestones called once on setSnapshots)

5. **Lifecycle:**
   - Init: empty (no execution loaded)
   - Load execution: setSnapshots(execution.snapshots) → computeMilestones automatically
   - Scrub: jumpToMilestone(index) updates currentSnapshotIndex
   - Exit: reset() clears all state

   ```typescript
   interface SnapshotScrubberProps {

     currentIndex: number;
     onScrub: (index: number) => void;
     onMilestoneHover?: (milestone: SnapshotMilestone) => void;

   ```

5. **Implementation:**
   ```typescript
   export function SnapshotScrubber({ milestones, currentIndex, onScrub }: Props) {
     const [dragging, setDragging] = useState(false);
     const trackRef = useRef<HTMLDivElement>(null);

     const handleMouseDown = () => setDragging(true);
     const handleMouseUp = () => setDragging(false);

     const handleMouseMove = (e: MouseEvent) => {
       if (!dragging || !trackRef.current) return;
       const rect = trackRef.current.getBoundingClientRect();
       const percent = Math.max(0, Math.min(1, (e.clientX - rect.left) / rect.width));
       let index = Math.floor(percent * (milestones.length - 1));

       // Snap to nearest milestone
       const nearest = milestones.reduce((prev, m) =>
         Math.abs(m.index - index) < Math.abs(prev.index - index) ? m : prev
       );
       onScrub(nearest.index);
     };

     return (
       <div className="flex flex-col gap-2 px-4 py-3 bg-muted rounded">
         {/* Track */}
         <div
           ref={trackRef}
           className="relative h-1 bg-secondary rounded cursor-pointer"
           onMouseDown={handleMouseDown}
           onMouseMove={handleMouseMove}
           onMouseLeave={() => setDragging(false)}
         >
           {/* Thumb */}
           <div
             className="absolute w-3 h-3 bg-primary rounded-full shadow-lg transform -translate-x-1/2 -translate-y-1/2 top-1/2 transition-left"
             style={{
               left: `${(currentIndex / Math.max(1, milestones.length - 1)) * 100}%`,
             }}
           />

           {/* Milestones */}
           {milestones.map((m) => (
             <button
               key={m.index}
               className="absolute w-2 h-2 bg-primary/60 rounded-full hover:bg-primary transition-colors"
               style={{
                 left: `${(m.index / Math.max(1, milestones.length - 1)) * 100}%`,
                 transform: 'translate(-50%, -50%)',
                 top: '50%',
               }}
               onClick={() => onScrub(m.index)}
               title={m.label}
             />
           ))}
         </div>

         {/* Label */}
         <div className="flex justify-between text-xs text-muted-foreground">
           <span>{milestones[currentIndex]?.label || 'Start'}</span>
           <span>{Math.round((currentIndex / Math.max(1, milestones.length - 1)) * 100)}%</span>
         </div>
       </div>
     );
   }
   ```

   - Logs panel subscribes to currentSnapshotIndex, scrolls to matching timestamp

7. **Accessibility:**

   - Keyboard navigation (Arrow keys)
   - Min hitbox 44x44px (mobile)
