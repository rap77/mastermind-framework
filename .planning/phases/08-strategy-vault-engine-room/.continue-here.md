## Decisions Made (All Locked)

### Dynamic DAG (for Phase 08-01 backend)
1. **Progressive Niche Expansion** — Macro view (all nichos as nuclear icons), drill-down on activation, Ghost state (40% opacity) for completed nichos
2. **Trace-Back Impact** — Error at brain (red glow) → niche border (red dotted) → edge (red blocked) → upstream to Master. Sibling brains pause (yellow).
3. **Pulse & Reveal** — Nuclear nodes expand on activation → interleaved in constellation → previous nichos fade to Ghost. Linear progression showing execution path.
4. **Snapshot Scrubbing** — Milestone-based replay (jump between state snapshots, not animate all events). Log sync with scrubber timeline. Engineering-tool feel.

### Phase 08 Features
5. **Focus-Driven Dynamic Console** — Logs follow active nicho. Click DAG node to isolate brain logs. Single react-virtuoso viewport. Brain-badge color per line.
6. **Context-Aware Focus Mode** — Auto on task start (sidebar→icons, idle dim 30%). [F] or Esc to override. Zero friction, full control.
7. **Smart-GFM** — react-markdown + GFM + custom component mapping (Recharts, DataTable, Prism, Lucide). Brains output pure Markdown, frontend renders rich visuals.
