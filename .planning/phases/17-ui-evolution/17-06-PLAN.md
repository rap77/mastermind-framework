- **Impact:** Task 1 OnboardingWizard component

## Rollback Plan

If onboarding wizard causes high drop-off:
1. Make onboarding skippable (skip button prominent)
2. Reduce to 3 steps (merge company + adapter steps)
3. Add "minimal setup" path (company name only, configure later)

If mobile gestures fail on real devices:
1. Remove swipe gestures, keep button actions
2. Add bottom sheet for mobile actions (Material Design pattern)
3. Use react-swipeable library (fallback to native events)

If accessibility audit fails:
1. Document known issues (create GitHub issues)
2. Add accessibility statement (partial compliance)
3. Prioritize fixes (Level A > AA > AAA)
