2. Scan for `[SYNC: BF-NN-ID]` tags using pattern `\[SYNC:\s+BF-(\d{2})-(\w+)\]`
3. For each tag found:
   a. Open the OWNER feed: `.planning/BRAIN-FEED-{NN}-{domain}.md` (NN from the tag)
