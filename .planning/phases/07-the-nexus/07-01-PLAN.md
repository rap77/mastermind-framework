<objective>
Validate and extend `GET /api/tasks/{id}/graph` to return a React Flow compatible payload.

Purpose: The existing endpoint returns `GraphNode` and `GraphEdge` models that are NOT compatible with React Flow's `Node`/`Edge` types — edges use `from_node`/`to` instead of `source`/`target`, and `layout_positions` is absent entirely. BE-02 requires `{ nodes[], edges[], layout_positions }` shape. This plan closes that gap at the FastAPI layer.

Output: Extended `TaskGraphResponse` model with `layout_positions: dict[str, dict] | None = None` field; `GraphEdge` serialized with `source`/`target` field names; pytest tests confirming the new contract.
</objective>
