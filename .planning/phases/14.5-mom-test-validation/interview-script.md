# Interview Script — Mom Test Validation

**Interviewer:** [Tu nombre]
**Interviewee:** [Nombre, empresa, rol]
**Date:** [Fecha]
**Duration:** [20-30 min]

## WARM-UP (5 min)

**Objetivo:** Establecer rapport, entender contexto

1. "Contame un poco sobre tu empresa — qué hacen, cuánta gente, tu rol."
2. "¿Cómo se define tu rol? ¿Sos más técnico, más negocio, o mixto?"
3. "¿Qué herramientas de IA usan en la empresa? (ChatGPT, Claude, agentes, etc.)"

## PAST BEHAVIOR — API Keys Configuration (10 min)

**Objetivo:** Entender CÓMO configuran actualmente, NO qué preferirían

4. **Pregunta clave:** "Contame la ÚLTIMA VEZ que tuviste que configurar una API key para un servicio nuevo (OpenAI, Anthropic, otro). ¿Cuándo fue? ¿Qué servicio era?"

   [Esperar respuesta, luego hacer follow-up basado en respuesta]

5. **Si configuró él/ella mismo:**
   - "¿Qué pasos seguiste? ¿Fuiste a la doc, a YouTube, preguntaste a alguien?"
   - "¿Qué fue lo más difícil de ese proceso?"
   - "¿Cuánto tiempo te tomó?"
   - "¿Hubo algún momento donde te trabaste o tuviste que buscar ayuda?"

6. **Si delegó (dev team, CTO, etc.):**
   - "¿Por qué decidiste no hacerlo vos mismo?"
   - "¿Qué haría falta para que lo hicieras vos mismo?"

7. **Pregunta sobre CLI:**
   - "¿Alguna vez usaste la terminal/CLI para configurar algo? ¿Cuál fue tu experiencia?"
   - "¿Qué sentiste al usarlo? ¿Frustración, confianza, miedo de romper algo?"

## PAIN POINTS — Deep Dive (10 min)

**Objetivo:** Descubrir PROBLEMAS reales, NO proponer soluciones

8. **Pregunta clave:** "¿Cuál fue la PARTE MÁS FRUSTRANTE de configurar esas API keys o herramientas?"

   [Si dice "nada fue frustrante", push gently:]
   - "¿En serio? Nada? ¿Ni siquiera encontrar dónde poner la key?"
   - "¿Qué mejoraría de esa experiencia?"

9. **Pregunta sobre onboarding:**
   - "¿Recordás el onboarding de alguna herramienta que te haya gustado mucho? ¿Por qué?"
   - "¿Y alguna que te haya parecido terrible? ¿Qué la hacía terrible?"

10. **Pregunta sobre GUI vs CLI:**
    - "Si tuvieras que configurar una nueva herramienta hoy, ¿preferirías una interfaz visual (clicks, botones) o comandos en la terminal? ¿Por qué?"

    [Si dice "depende":]
    - "¿Depende de qué? ¿Cuándo preferirías uno vs el otro?"

## WRAP-UP (5 min)

**Objetivo:** Capturar insights finales

11. **Pregunta abierta:** "¿Hay algo sobre configurar herramientas de IA que te gustaría que fuera más fácil pero hoy es difícil?"

12. **Pregunta para ellos:** "¿Tenés alguna pregunta para mí sobre lo que estoy explorando?"

13. **Cierre:** "Gracias por tu tiempo. ¿Te gustaría ver los resultados de esta investigación cuando los tenga?"

---

## CRITICAL RULES

❌ **NO decir:**
- "Estoy construyendo un onboarding GUI para MasterMind"
- "¿Te gustaría usar una interfaz visual en lugar de CLI?"
- "¿Usarías esta herramienta si la hiciera?"

✅ **SÍ decir:**
- "Contame sobre la última vez que configuraste..."
- "¿Qué fue lo más difícil de...?"
- "¿Cómo te sentiste al usar...?"
