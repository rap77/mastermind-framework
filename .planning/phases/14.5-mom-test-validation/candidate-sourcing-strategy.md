# Candidate Sourcing Strategy — Mom Test Validation

**Objetivo:** Identificar 3 CEOs técnicos de LATAM para entrevistar
**Perfil:** CEO/fundador (5-50 empleados), usa IA, configura API keys

## Fuentes de Candidatos (Priorizadas)

### 1. LinkedIn (MEJOR opción)

**Search queries:**
```
CEO OR Founder OR CTO
Current company: Startup (5-50 employees)
Location: Argentina, Mexico, Colombia, Chile
Keywords: AI, artificial intelligence, machine learning, automation
```

**Filtros adicionales:**
- "Show all founders" → Filtra por no-employee founders
- "Open to connect" → Más receptivos
- "Activity" → Últimos 30 días activos

## Outreach Template

**Asunto:** Research interview — Tech CEO feedback (15 min)

**Mensaje:**

```
Hola [Nombre],

Soy [Tu nombre], founder de MasterMind (framework de orquestación de agentes de IA para LATAM).

Estoy haciendo research sobre cómo los CEOs técnicos configuran herramientas de IA hoy en día — específicamente el proceso de setup de API keys y onboarding inicial.

Me gustaría entrevistarte por 15-20 min (Zoom o Google Meet). NO es una venta — es research puro para validar un problema que muchos mencionan.

Tu perfil encaja porque:
- [Razón específica: ej. "Fuiste CTO antes de founder", "Tu startup usa IA", etc.]

A cambio, te comparto los insights anónimos del estudio (3 entrevistas total) + acceso anticipado a lo que construyamos.

¿Te interesa? Si no, ¿conocés a alguien que pueda encajar?

Gracias,
[Tu nombre]
[LinkedIn/Twitter]
```

## Criterios de Filtrado

**MUST HAVE (3/3):**
1. ✅ CEO/Co-founder/CTO con equity
2. ✅ Empresa 5-50 empleados
3. ✅ Usa herramientas de IA (ChatGPT, Claude, APIs)

**NICE TO HAVE (2+):**
- ⚡ Background técnico (sabe programar)
- ⚡ Configuró API keys él/ella mismo (últimos 6 meses)
- ⚡ Based en LATAM (Argentina, Mexico, Colombia, Chile)
- ⚡ Habla español o inglés
