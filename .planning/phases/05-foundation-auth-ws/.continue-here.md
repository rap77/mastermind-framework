<context>

**El problema:**
Immer middleware congela el estado para prevenir mutaciones accidentales. `get()` retorna un frozen proxy. La línea 28 intenta `push()` directamente sobre ese objeto congelado, causando "Cannot add property 1, object is not extensible".

**El fix:**
```typescript
// Antes (❌)
updateBrain: (brain) => {
  get()._queue.push(brain)  // Frozen proxy error
  // ...
}

// Después (✅)
updateBrain: (brain) => {
  set(state => {
    state._queue.push(brain)  // state es mutable draft aquí
    // ...
  })
}
```
