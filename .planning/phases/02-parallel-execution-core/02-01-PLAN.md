  <behavior>

  <action>
Create mastermind_cli/orchestrator/dependency_resolver.py with:

1. **ExecutionGraph model** (Pydantic):
   ```python
   from pydantic import BaseModel
   from typing import List, Dict, Set

   class ExecutionLevel(BaseModel):
       wave_number: int
       brain_ids: List[str]

   class ExecutionGraph(BaseModel):
       levels: List[ExecutionLevel]
       total_brains: int
       max_parallelism: int  # Max brains in any wave
   ```

2. **DependencyResolver class**:
   ```python
   class DependencyResolver:
       def __init__(self, brain_registry: BrainRegistry):
           self.registry = brain_registry

       async def resolve(self, flow: FlowConfig) -> ExecutionGraph:
           # 1. Get topological order from flow
           # 2. Build wave structure:
           #    - Wave 1: All brains with zero dependencies
           #    - Wave N: Brains whose dependencies are all in waves 1..N-1
           # 3. Validate all brain_ids exist in registry
           # 4. Return ExecutionGraph with waves
   ```

3. **Wave building algorithm**:
   - Start with topological order from flow.get_execution_order()
   - For each brain in order: find maximum wave level of dependencies
   - Assign brain to wave = max_dep_wave + 1
   - Group brains by wave number
   - Calculate max_parallelism = max(len(wave.brain_ids) for wave in levels)

4. **Validation**:
   - Check all brain_ids exist in self.registry.list_brains()
   - Raise ValueError with clear message for unknown brains

Use asyncio for async method (future-proofing for database lookups).
  </action>
