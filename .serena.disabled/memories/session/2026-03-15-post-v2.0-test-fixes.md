## Patterns Discovered

### Mypy Strict Mode Pattern
When using incremental dict assignment with mypy strict:
```python
# DON'T (type inference issues)
normalized["field"] = some_value  # Type inferred from first assignment

# DO (explicit types)
field_value: DesiredType = default
normalized["field"] = field_value
```

### Brief Validation Pattern
All Brief instances require:
- Minimum 3 words (field validator)
- Minimum 10 characters (field constraint)
- Use: `"Build a comprehensive CRM system"` ✓
- Avoid: `"Test brief"` ✗ (2 words)
