### Key Decisions Made
1. **Type Safety FIRST** - Pydantic models enable everything else
2. **asyncio over distributed queues** - Single-host deployment doesn't need Celery/RQ
3. **FastAPI + HTMX over React** - Simpler, no build step, sufficient for dashboard
4. **ExperienceRecord with embedding_stub** - v3.0-ready without premature optimization
5. **Backward compatibility critical** - All 23 existing brains must work

### Research Warnings Identified
- **Pitfall #5:** Multi-orchestrator race (global state prevents concurrent sessions)
- **Pitfall #6:** MCP integration bottleneck (concurrent requests may rate limit)
- **Pitfall #2:** False parallelism (mixing sync/async kills performance)
- **Pitfall #4:** Auth state schism (CLI and web using different auth systems)
