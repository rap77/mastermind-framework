# MasterMind Framework - NotebookLM Naming Convention

## Standard Format

**All notebooks must use the prefix `[CEREBRO]` not `[MM]`:**

```
[CEREBRO] {Brain Name} - {Niche}
```

## Examples

```
[CEREBRO] Product Strategy - Software Development
[CEREBRO] Growth & Data - Software Development
[CEREBRO] UX Research - Software Development
```
