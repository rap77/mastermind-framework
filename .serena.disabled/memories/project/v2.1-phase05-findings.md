## Patterns Established

1. **SSR-safe WebSocket init** — `typeof window` guard in connect(), never module-level
2. **RAF batching** — Queue updates, drain before paint (prevents 24-event freeze)
3. **Targeted selectors** — `useBrainState(id)` prevents cascade re-renders
4. **Zod before Zustand** — Validate WS events before store updates
5. **Module-level stores** — wsStore and brainStore survive navigation

## Pitfalls Avoided

1. **React Flow CSS** — Must be in `@layer base` of globals.css (not tsx imports)
2. **Next.js 16 cookies()** — Now async, must `await cookies()`
3. **CORS with credentials** — Wildcard prohibited when `allow_credentials=True`
4. **Immer mutations** — All state changes must be inside `set()` callback
