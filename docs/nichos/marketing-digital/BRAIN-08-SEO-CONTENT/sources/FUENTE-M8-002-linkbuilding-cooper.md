# FUENTE-M8-002: Link Building Strategies — Jon Cooper

## Tesis Central

> Link building NO es pedir links, es crear content que la gente naturalmente quiere linkear. El outreach directo ("podés linkearme?") tiene < 5% success rate porque es transactional: vos querés algo (link) sin dar valor a cambio. El link building efectivo es relationship-based: construís relaciones con otros creators, les das value upfront, y los links vienen naturalmente. Las estrategias más efectivas son: (1) **Skyscraper** — creás algo mejor que lo que ya está linkeado, (2) **Broken Link Building** — ayudás a webmasters a encontrar links rotos, (3) **Digital PR** — creás newsworthy content que journals/media citan, (4) **Guest Posting** — aportás value a otros sites a cambio de links. La clave es: NO perseguí links, perseguí relationships. Los links son el byproduct de crear value y construir trust.

## 1. Principios Fundamentales

> **P1: Link quantity < link quality (hoy más que nunca)**
- 1 link de NY Times vale más que 1000 links de sitios spam
- Google's Penguin mató la estrategia de cantidad
- Domain Authority (DA) del site que linkea importa MUCHO
> *Fuente: "Quality Over Quantity" (Cooper, 2023)*

> **P2: Link begging (outreach random) = 99% rejection rate**
- "Hi, I loved your post, can you link to mine?" = deleted
- Outreach sin value = spam
- Webmasters reciben 100+ emails así por día
> *Fuente: "Outreach Reality" (Cooper, 2023)*

> **P3: Links vienen de value, no de asking**
- Si tu content es amazing, people WILL link
- Si tu content es mediocre, no amount de outreach va a ayudar
- Linkable assets = content con original research/data
> *Fuente: "Value-First Link Building" (Cooper, 2023)*

> **P4: Relationships > transactions**
- Construí relationship con webmasters PRIMERO
- Dales value (content, insights, data) ANTES de pedir nada
- Cuando pidás algo después, success rate es 20-50%
> *Fuente: "Relationship Building" (Cooper, 2023)*

> **P5: Anchor text natural > exact match**
- Exact match anchor text ("best SEO tools") = over-optimization
- Natural anchor text ("click here", "this guide", brand name) = safe
- Google penaliza over-optimized anchor text
> *Fuente: "Anchor Text Safety" (Cooper, 2023)*

## 2. Frameworks y Metodologías

### Framework 1: Skyscraper Technique

**Fuente:** "Link Building Guide" (Cooper, 2023)
**Propósito:** Conquistar links mejorando content existente.
**Cuándo usar:** Cuando competitors tienen links para content que podés mejorar.

**Step 1: Find Link-Worthy Content**

```yaml
Herramientas:
  - Ahrefs → Content Explorer
  - BuzzSumo
  - SEMrush

Proceso:
  1. Enter your target keyword
  2. Sort by backlinks (descending)
  3. Identify content with 50+ backlinks
  4. Analyze: ¿es este content mejorable?
```

**Step 2: Create Something Better**

```yaml
Ways to Improve:
  - Make it longer (2000+ words vs 500)
  - Make it fresher (2024 data vs 2020)
  - Make it visual (charts, infographics vs text only)
  - Make it actionable (checklists, templates vs theory)
  - Make it original (unique data vs curated content)

Benchmark: Tu content debe ser 2x mejor que el original.
```

**Step 3: Outreach to People Linking to Original**

```yaml
Find Linkers:
  - Ahrefs → Backlinks → Export list
  - Filter by: DR (Domain Rating) 40+, traffic 5K+

Outreach Email Template:
  Subject: "Better version of [content they linked]"

  Hi [Name],

  I noticed you linked to [original content] in your post on [topic].
  Great resource!

  I just published a more comprehensive/updated version:
  [your content]

  It includes [unique feature: original data/study/visualization]
  that the original doesn't have.

  Thought you might want to check it out. If you find it valuable,
  feel free to update your link.

  No pressure, just sharing something better for your readers.

  Best,
  [Your Name]

Success Rate: 10-20% (10x better than cold outreach)
```

### Framework 2: Broken Link Building

**Fuente:** "Broken Link Building" (Cooper, 2023)
**Propósito:** Ayudar webmasters a encontrar links rotos.
**Cuándo usar:** Cuando encontrás broken links en tu niche.

**Step 1: Find Broken Links**

```yaml
Herramientas:
  - Ahrefs → Backlinks → Broken
  - Screaming Frog → Custom → Response Codes
  - Check My Links (Chrome extension)

Target Sites:
  - Resource pages in your niche
  - Blogs with older posts (more likely broken links)
  - Educational/government sites (.edu, .gov)
```

**Step 2: Create Replacement Content**

```yaml
Create content that replaces the broken link:
  - Similar topic
  - Better quality
  - More comprehensive
  - Up-to-date

Example:
  - Broken: "2020 SEO Guide" (outdated)
  - Your: "2024 SEO Guide" (fresh, comprehensive)
```

**Step 3: Outreach to Webmaster**

```yaml
Outreach Email Template:
  Subject: "Broken link on [page]"

  Hi [Name],

  I was reading your excellent resource page on [topic].
  Really helpful!

  I noticed a broken link:
  [URL of broken link] → 404 error

  I actually created a resource that would be a great replacement:
  [your content]

  It covers [specific topic] and includes [unique feature].

  Would you consider updating the link? Let me know if you need
  any other resources on this topic.

  Thanks for creating such a helpful page!

  Best,
  [Your Name]

Success Rate: 20-30% (high because you're helping, not asking)
```

### Framework 3: Digital PR

**Fuente:** "Digital PR Guide" (Cooper, 2023)
**Propósito:** Conseguir links de news/media sites.
**Cuándo usar:** Cuando tenés newsworthy content.

**Types of Content That Get Media Links:**

| Type | Example | Why it works |
|------|---------|--------------|
| Original Research | "Industry Survey 2024" | Journalists love data |
| Studies | "We analyzed 1M websites" | Citable findings |
| Infographics | "State of Marketing 2024" | Visual, shareable |
| Tools | "Free SEO Calculator" | Useful resource |
| Listicles | "50 Marketing Statistics" | Easy to reference |
| Predictions | "5 Trends for 2025" | Timely, quotable |

**Digital PR Process:**

```yaml
Step 1: Create Newsworthy Content
  - Original research (survey, study, analysis)
  - Data-driven insights (statistics, benchmarks)
  - Visual elements (infographics, charts)
  - Timely topic (trends, predictions)

Step 2: Build Media List
  - Identify journalists covering your niche
  - Twitter lists of industry journalists
  - HARO (Help a Reporter Out)
  - Press release distribution

Step 3: Pitch Journalists

  Pitch Email Template:
  Subject: "Story idea: [newsworthy angle]"

  Hi [Journalist Name],

  I saw your recent piece on [topic]. Great insights on [specific point].

  I have a story idea that might interest your [publication] readers:

  [Newsworthy angle: original data/timely trend]

  We just completed [study/survey] with [key finding].

  For example, we found that [statistic] which challenges the common
  belief that [misconception].

  Full report here: [link]

  Would you be interested in covering this? Happy to provide more data
  or connect you with our [expert].

  Best,
  [Your Name]

Success Rate: 1-5% (but links are HIGH quality: NYT, Forbes, etc.)
```

### Framework 4: Guest Posting

**Fuente:** "Guest Posting Guide" (Cooper, 2023)
**Propósito:** Conseguir links a través de guest posts.
**Cuándo usar:** Para construir authority y relationships.

**Guest Posting Process:**

```yaml
Step 1: Find Guest Post Opportunities
  - Google: "[niche] + write for us"
  - Google: "[niche] + guest post"
  - Analyze competitors' guest posts (Ahrefs)
  - Quality sites: DA 40+, traffic 10K+

Step 2: Pitch Topics

  Pitch Email Template:
  Subject: "Guest post idea for [site]"

  Hi [Editor Name],

  I've been following [site] for a while. Love your recent post on [topic]!

  I'd love to contribute a guest post on:
  [Topic proposal: specific, relevant to their audience]

  Here's what I'd cover:
  - [Point 1]
  - [Point 2]
  - [Point 3]

  I'm [your credentials: experience, publications].

  Would this be of interest? I can also send other topic ideas if this one
  doesn't fit.

  Best,
  [Your Name]

Step 3: Write Amazing Content
  - Follow their guidelines (word count, formatting)
  - Include original insights (not generic advice)
  - Add relevant links (1-2 to your site, 5+ to authority sites)
  - Include author bio with link to your site

Step 4: Follow Up
  - Share on social media (tag the publication)
  - Respond to comments
  - Build relationship for future guest posts
```

**Guest Post Quality Standards:**

| Standard | Good | Bad |
|----------|------|-----|
| Length | 1000-1500 words | < 500 words |
| Originality | Unique insights | Rehashed content |
| Links | 1-2 to your site, 5+ to authorities | 5+ to your site |
| Credentials | Demonstrated expertise | No credentials shown |
| Promotion | Shared on social media | No promotion |

## 3. Modelos Mentales

### Modelo 1: La Ecuación de Link Building

```
Link Acquisition =
  (Value × Relationship × Relevance)
  ───────────────────────────────────
        Effort Required
```

**Key insight:** High value + strong relationship = links with less effort.

### Modelo 2: El Funnel de Link Building

```
Outreach (100 emails)
      ↓
Response Rate (20% = 20 responses)
      ↓
Link Acquisition (50% of responses = 10 links)
      ↓
Links Stick (90% remain after 6 months)
```

**Key insight:** De 100 emails, esperá 5-10 links de calidad.

### Modelo 3: La Matriz de Estrategias de Link Building

```
           │ Easy  │ Difficult │
────────────────────────────────────

           │Link  │             │
────────────────────────────────────

           └──────┴─────────────┘
```

**Key insight:** Priorizá Skyscraper y Digital PR (High Value).

## 4. Criterios de Decisión

### Decisión 1: ¿Skyscraper o Broken Link Building?

**Usá Skyscraper si:**
- Tenés capacity para crear content mejor que competitors
- El content original tiene potencial (good topic, outdated execution)
- Tu competition tiene links que podés "steal"
- Buscás links de alta calidad

**Usá Broken Link Building si:**
- Encontraste broken links relevantes
- Podés crear replacement content rápidamente
- Target sites son responsive (smaller sites, personal blogs)
- Buscás quick wins (20-30% success rate)

### Decisión 2:¿Outreach Directo o Relationship-Building?

**Outreach Directo si:**
- Time-sensitive campaign
- One-off link building
- Target site is large/impersonal

**Relationship-Building si:**
- Long-term SEO strategy
- Querés recurring links from same sites
- Target site is medium/small (personal connection matters)

## 5. Anti-patrones

### Anti-patrón 1: "Mass Outreach (Spamming)"

**Problema:** Enviás el mismo email genérico a 1000 sites.

**Consecuencias:**
- Marked as spam
- Reputation damage
- Future emails blocked

**Solución:** Personalized outreach, 20-50 targeted emails per campaign.

### Anti-patrón 2: "Buying Links"

**Problema:** Comprás links de Fiverr, marketplaces.

**Consecuencias:**
- Google penalty (Penguin)
- Lost rankings when discovered
- Reputation damage

**Solución:** Earn links through value creation.

### Anti-patrón 3: "Exact Match Anchor Text Overload"

**Problema:** 10 links con "best SEO tools" anchor text.

**Consecuencias:**
- Over-optimization penalty
- Rankings drop
- Manual review risk

**Solución:** Natural anchor text (brand name, URL, generic).

## 6. Checklists de Implementación

### Checklist de Link Building Campaign

**Planning:**
- [ ] Target sites identified (DA 40+, relevant)
- [ ] Link goals defined (10 links/month?)
- [ ] Content assets created (linkable content)
- [ ] Outreach list prepared

**Outreach:**
- [ ] Personalized emails written
- [ ] Email templates tested (subject lines, CTAs)
- [ ] Follow-up sequence planned
- [ ] Tracking set up (open rates, response rates)

**Follow-up:**
- [ ] Responses tracked in spreadsheet
- [ ] Links acquired documented
- [ ] Relationships nurtured (future opportunities)
- [ ] Failed outreach analyzed (why no response?)

**Quality Control:**
- [ ] Links are dofollow
- [ ] Anchor text is natural
- [ ] Link context is relevant
- [ ] Link sticks (not removed after 1 month)
