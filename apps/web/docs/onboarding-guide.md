# MasterMind Onboarding Guide

Welcome to MasterMind! This guide will walk you through the onboarding process and help you get started with your AI-powered command center.

## Getting Started

When you first access MasterMind, you'll be guided through a 3-step onboarding process:

### Step 1: Welcome

- Learn about MasterMind's key features
- Discover the **Cmd+K** (Mac) or **Ctrl+K** (Windows/Linux) shortcut for the command palette
- Click "Get Started" to begin setup

### Step 2: Company & Adapter Setup

**Company Information:**
- Enter your company name
- Select your industry
- Choose your primary goal (Product Strategy, UX Research, UI Design, etc.)

**AI Adapter Selection:**
- **OpenAI**: GPT-4, GPT-4 Turbo (requires API key)
- **Anthropic**: Claude 3 Opus, Sonnet (requires API key)
- **Local LLM**: Ollama, LocalAI (requires base URL)

### Step 3: Validation

- Review your setup
- Test your adapter connection
- Complete onboarding

## Keyboard Shortcuts

MasterMind is designed for power users who prefer keyboard navigation:

| Shortcut | Action |
|----------|--------|
| `Cmd+K` / `Ctrl+K` | Open command palette |
| `↑` / `↓` | Navigate commands |
| `Enter` | Execute selected command |
| `Esc` | Close dialog/palette |

## Command Palette

The command palette (`Cmd+K` / `Ctrl+K`) is your primary interface for:

- **Navigation**: Quick access to War Room, Nexus, Strategy Vault, Engine Room
- **Brains**: Trigger 24 expert AI agents across 6 domains
- **Actions**: Create company, switch company, export data
- **Settings**: API keys, brain configuration, user profile

### Brain Domains

Brains are organized into 6 domains to reduce cognitive load:

1. **Product Strategy** (4 brains): Product Vision, Market Fit, Roadmap Planner, Prioritization
2. **UX Research** (4 brains): User Research, Personas, Journey Map, Usability Testing
3. **UI Design** (4 brains): Design System, Visual Design, Prototyping, Accessibility
4. **Frontend** (4 brains): Frontend Architecture, State Management, Performance, Testing
5. **Backend** (4 brains): API Design, Database Design, Authentication, Scalability
6. **QA/DevOps** (4 brains): QA Strategy, CI/CD, Monitoring, Incident Response

## Mobile Features

MasterMind is fully responsive and optimized for mobile devices:

### Bottom Navigation

On mobile devices (< 768px), you'll see a bottom navigation bar with 4 main sections:

- **War Room**: Your command center dashboard
- **Nexus**: Browse and trigger expert brains
- **Strategy**: Planning and prioritization tools
- **Settings**: Configure your preferences

### Touch Targets

All interactive elements meet WCAG 2.1 accessibility standards with minimum touch target sizes of 44x44px.

## Troubleshooting

### Onboarding Issues

**Problem**: Can't complete onboarding
- **Solution**: Ensure all required fields are filled (company name, industry, goal, API key/base URL)

**Problem**: Adapter validation fails
- **Solution**: Check your API key or base URL. For local LLMs, verify your Ollama/LocalAI service is running.

### Command Palette Issues

**Problem**: `Cmd+K` doesn't open command palette
- **Solution**: On Chrome Mac, `Cmd+K` may open the dev console. Use `Cmd+Shift+K` as an alternative, or change your browser shortcuts.

**Problem**: Commands not found
- **Solution**: Use broader search terms. The fuzzy search works with partial matches (e.g., "strat" → "Strategy Vault").
