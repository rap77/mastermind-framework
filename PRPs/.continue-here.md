- JWT auth with refresh token rotation (access 30min, refresh 24h)
- WebSocket with exponential backoff + adaptive polling fallback
- D3.js for graph (not React Flow) - layered layout, simpler API

**PRP-03-00 Architecture (Simplification Cascade):**
- Pure Function Architecture: "Input → Output, no shared state"
- Reduces complexity by 80% (40+ files → 8 files, ~3000 LOC → ~600 LOC)
- Stateless coordinator: per-request instances (multi-user safe)
- MCP sequential by wave (no rate limit issues)
- Legacy wrapper for backward compatibility
